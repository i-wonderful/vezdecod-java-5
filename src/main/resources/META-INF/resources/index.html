Vezdecod. Task 5. Java.

Этот мкс для задания 5 java. Написан на quarkus. Используется postgres, hibernate, классический rest. Интеграция с сервисом https://jservice.io. В задании 3 добавлено кеширование. В задании 4 добавлена игра. В задании 5 добавлены тесты. Для удобства проверки и наглядности добавлен сваггер.

Рабочие урлы:

  • ui swagger потыкать рест
  • GET /question/random получить рандомный вопрос
  • POST /question/check проверить ответ на вопрос (формат отправляемых данных см.в задании)
  • POST /game создать игру
  • GET /game/{game_id}/{question_number} получить вопрос
  • POST /game/{game_id}/{question_number}/check проверить вопро
  • POST /game/{game_id}/{question_number}/finish вывести все отвеченные вопросы и флаг верного ответа
