Vezdecod. Task 3. Java.

Этот мкс для задания 3 java. Написан на quarkus. Используется postgres, hibernate, классический rest. Для удобства проверки и наглядности добавлен сваггер.

Рабочие урлы:

  • ui swagger потыкать рест
  • GET /question/random получить рандомный вопрос
  • POST /question/check проверить ответ на вопрос (формат отправляемых данных см.в задании)

Почему quarkus? Потому что он прекрасен и воплощает все лучшее в современной java разработке. Быстрый и удобный, умеет билдится в нейтив сборку. Активно развивается, с обширной документацией. Так почему бы его не заюзать и тут, ведь мы здесь собратись just for fun.
